# 퇴직금 운용 계획

기준일: 2026-03-10

## 1. 목적

- 과학기술공제 퇴직금의 코어는 예금형 복리 4.5%를 유지한다.
- 예금형의 안정성을 살리면서 일부만 성장자산으로 분산한다.
- 한 번에 전환하지 않고, 매 실행 시점마다 먼저 진단한 뒤 단계적으로 리빌딩한다.

## 2. 현재 전제

- 현재 적립금: 33,787,210원
- 연간 추가 납입: 9,000,000원
- 현재 상태: 예금형 100%
- 예금형 수익률: 복리 4.5%

## 3. 기본 전략

- 예금형 70%
- 운용형 30%

의미:

- 예금형 70%는 계속 유지하는 코어 자산이다.
- 운용형 30%만 주식형 중심으로 분할 매수한다.
- 아직 매수하지 않은 금액은 현금 방치가 아니라 예금형 4.5%로 계속 운용된다.

## 4. 최종 목표 포트폴리오

### 1안: 기본 추천안

- 예금형 70%
- 미래에셋MSCIAC월드인덱스증권자투자신탁(주식)(UH)종류C-P2e 15%
- 유리필라델피아반도체인덱스증권자투자신탁UH[주식]ClassC-P1 10%
- 삼성퇴직연금KOSPI200증권자투자신탁 1[주식]_C 5%

예상 연평균 수익률:

- 보수적 기대: 4.8%
- 기준 기대: 5.2%
- 낙관 시나리오: 6.3%

해석:

- 예금형 4.5%가 전체의 70%를 받쳐 준다.
- 나머지 30%는 글로벌 코어와 반도체, 한국 대형주로 분산한다.
- 퇴직성 자금 기준으로 가장 균형적이다.

### 2안: 수익 우선안

- 예금형 70%
- 미래에셋MSCIAC월드인덱스증권자투자신탁(주식)(UH)종류C-P2e 10%
- 유리필라델피아반도체인덱스증권자투자신탁UH[주식]ClassC-P1 15%
- 한국투자글로벌AI&반도체TOP10증권자투자신탁UH(주식)(C-Re) 5%

예상 연평균 수익률:

- 보수적 기대: 5.0%
- 기준 기대: 5.8%
- 낙관 시나리오: 7.2%

해석:

- 1안보다 기대수익은 높다.
- 반도체와 AI 편중이 커져 변동성도 더 높다.
- 기본 운용안으로는 1안을 우선 채택한다.

## 8. 매 실행 전 진단 절차

다음 분할 매수 전에는 아래 순서로 먼저 점검한다.

### 1. 금리 점검

- 과학기술공제 예금형 4.5% 조건이 유지되는지 확인
- 한국 기준금리 변화 여부 확인
- 채권금리 급변 여부 확인

판단:

- 예금형 4.5%가 유지되면 코어 70% 전략 유지
- 예금형 금리가 크게 낮아지면 전략 재검토

### 2. 시장 점검

- 미국 대형주와 반도체 지수의 최근 1개월 흐름 확인
- 한국 KOSPI 흐름 확인
- 시장이 단기 급등했는지, 급락했는지 확인

판단:

- 보합 또는 완만한 상승: 계획대로 진행
- 단기 급등: 분할 매수 유지
- 급락: 코어 자산 비중을 우선 채우며 계획 조정

### 3. 지정학 점검

- 전쟁, 원유 급등, 환율 급등, 금융위기성 뉴스 확인
- 일시적 뉴스인지 구조적 충격인지 구분

판단:

- 단기 뉴스성 충격: 원칙 유지
- 시스템 리스크 확대: 신규 매수 비중 축소 또는 한 차례 보류 검토

### 4. 상품 점검

- 해당 펀드가 계속 판매 중인지 확인
- 총보수 변경 여부 확인
- 최근 성과보다 역할이 유지되는지 확인

판단:

- 코어 펀드는 역할 유지 여부가 핵심
- 테마 펀드는 과열 시 비중 확대 금지

## 9. 시장 급변 시 대응 규칙

### 5%~10% 급락

- 계획대로 진행
- 분할 매수 유지

### 10%~15% 급락

- 추가 매수는 가능
- 다만 테마형보다 MSCI AC World 같은 코어형을 우선 매수

### 20% 이상 급락

- 예금형 70% 원칙은 유지
- 목표 비중 안에서만 리밸런싱
- 공포에 의한 일괄 매수 또는 일괄 환매 금지

### 전쟁, 유가 급등, 환율 급등

- 에너지, 금, 방산 테마로 즉흥 전환하지 않는다.
- 단기 헤드라인보다 기존 목표 비중 유지 여부를 먼저 본다.
- 충격이 2주 이상 지속되고 시장 구조가 변하면 재진단 후 계획 수정

## 10. 실행 원칙

- 목표 비중보다 감정이 앞서면 매수 보류
- 수익률만 보고 테마형 비중 확대 금지
- 분기 1회 또는 목표 비중에서 5%p 이상 이탈 시 리밸런싱 검토
- 예금형 코어를 깨는 결정은 별도 재검토 후 진행
